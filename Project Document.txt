Road Accident Analysis

This project presents an end-to-end Road Accident Analysis using Power BI, including data cleaning, ETL (Power Query), star-schema data modeling, DAX measures, KPIs, and interactive dashboard visuals.
The goal is to identify trends, accident hotspots, severity patterns, and the top contributing factors to support data-driven decision-making.
